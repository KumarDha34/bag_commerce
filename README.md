## 🎯 Overview

**Bag** is a premium e-commerce platform specializing in handcrafted, embroidered bags. The website showcases products, allows users to browse collections, manage their shopping cart, save favorite items to wishlist, place orders, and track their purchase history.

This frontend application is designed to work with a Django REST Framework backend API, providing a complete headless e-commerce solution.

### Key Highlights

- ✅ **100% Responsive** - Works perfectly on desktop, tablet, and mobile devices
- ✅ **Mobile-First Design** - Hamburger menu, touch-friendly buttons, optimized layouts
- ✅ **Complete User Journey** - From browsing to checkout to order tracking
- ✅ **Modern UI/UX** - Clean, elegant design with smooth animations
- ✅ **Performance Optimized** - Fast loading, lazy loading where applicable

## ✨ Features

### 🛍️ Shopping Experience
| Feature | Description |
|---------|-------------|
| Product Browsing | Grid and list view options with pagination |
| Advanced Filtering | Filter by category, price range, and customer rating |
| Sorting | Sort by newest, price (low-high/high-low) |
| Quick View | Product details modal without leaving the page |
| Product Details Page | Complete product information with images and meta data |

### 👤 User Account Management
| Feature | Description |
|---------|-------------|
| Registration | Create account with password strength validation |
| Login | Secure authentication with remember me option |
| Profile Management | Update personal information and profile picture |
| Password Change | Change password with current password verification |
| Dashboard | Overview of orders, wishlist, and spending analytics |

### 🛒 Cart & Checkout
| Feature | Description |
|---------|-------------|
| Add to Cart | Single-click add with quantity selection |
| Cart Management | Update quantities, remove items, clear entire cart |
| Price Calculation | Real-time subtotal, discount, and total calculation |
| Guest Cart | Cart persistence for non-logged-in users |
| Order Summary | Clear breakdown of charges before checkout |

### ❤️ Wishlist
| Feature | Description |
|---------|-------------|
| Save for Later | Add/remove products from wishlist |
| Move to Cart | Transfer items directly from wishlist to cart |
| Guest Wishlist | Local storage support for non-authenticated users |

### 📦 Order Management
| Feature | Description |
|---------|-------------|
| Order History | View all past orders with status tracking |
| Order Details | Detailed view of each order with items and shipping info |
| Order Cancellation | Cancel eligible orders |
| Status Tracking | Real-time order status (pending, confirmed, shipped, delivered) |

### 📱 Mobile Features
- Responsive navigation with hamburger menu
- Touch-friendly buttons and form inputs
- Optimized layouts for small screens
- Bottom navigation for easy access
- Smooth scrolling and animations
